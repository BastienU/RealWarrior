# Real Warrior

A game developed in **C#** with **.NET**, playable directly from the console.  

---

## Prerequisites

Before launching the game, make sure you have installed:

### .NET SDK
Download :  
https://dotnet.microsoft.com/download

Check the installation :
```bash
dotnet --version
```

## IDE
If you are using Visual Studio Code, you can install the following extensions :

### C# Dev Kit (Microsoft)

If you are using Visual Studio 2022, during installation enable support for :

### .NET Desktop Development (Développement .NET Desktop)
### Cross-platform Development with .NET (Développement multiplateforme avec .NET)

Then open the .sln file or the project folder.

## Start the game

To run the program, first move to the correct directory :
```bash
cd the location where you saved the "Real Warrior" directory + "/Jeu_Combat"
```
Then execute the following command :
```bash
dotnet run
```

I hope you will have as much fun playing the game as I had creating it.
